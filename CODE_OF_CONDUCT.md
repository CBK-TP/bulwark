# Code of Conduct

This project is for practical security work around Minecraft servers.

Keep discussion technical and useful:

- be direct without being abusive;
- assume false positives are worth fixing;
- do not shame server owners for insecure configs;
- do not post exploit details when private disclosure is more appropriate;
- do not publish secrets, leaked plugins or customer data.

Maintainers may close, edit or hide comments that make the project less safe or less usable.
